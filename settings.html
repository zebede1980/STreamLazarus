Server Plugin Checking…
The server plugin routes generate calls through a local proxy that keeps the upstream API call alive even when iOS drops the TCP connection — the only way to recover responses after a full tab kill. To install: copy the server-plugin/ folder from this extension into SillyTavern's plugins/stream-lazarus/ directory, set enableServerPlugins: true in config.yaml, then restart ST.
