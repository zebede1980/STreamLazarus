Stream Lazarus
Recovers AI responses when iOS drops the browser connection mid-generation.
When enabled, recovered text is inserted into the chat immediately without prompting. Leave off until you have confirmed recovery is reliable in your setup.
When enabled (default), Stream Lazarus only activates on phones and tablets. Uncheck to also run on desktop browsers.
Proxy Container Checking…
The stream-lazarus Docker container must be running and Nginx must proxy to it (port 3000) instead of directly to SillyTavern. See the project README for the docker-compose snippet.
